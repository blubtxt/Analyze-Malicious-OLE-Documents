
# Overview:
Different stuff to analyze malicious OLE files. With focus on .docx.
For example Trojan:JS/Certor.A

## unpack.py:
A Python script to unpack the OLE files in a .docx 

## base64_decode.py:
A Python script to decode data encoded with Base64
